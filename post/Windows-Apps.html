Windows 常用软件

2019年8月23日
| Windows
| 阅读
文章目录

这里是一些我常用的 Windows 软件，可以让你相对轻松的从几款知名或者优秀的软件中，寻找到最适合自己的。

本文最后更新于 ：2019年8月23日

视频播放器

PotPlayer：全格式，支持硬件加速。

杀毒软件

火绒：强悍、轻巧、干净。

磁盘管理

DiskGenius：老牌国产软件，专注数据恢复与磁盘管理，中文界面。

文本编辑器

Notepad++：开源软件，知名度极高，来自台湾，推荐。 Visual Studio Code：微软出品。

办公软件

Office Tool Plus ：帮助部署 Office 软件。

网页浏览器

Google Chrome：自从 Google 推出浏览器以来，Chrome 就以势不可挡的趋势占领着市场，超越对手。如果你想拥有一个浏览快速、更新频繁的网络体验，选它没错。

CentBrowser ：基于Chromium，可视为Chrome定制版。

Mozilla Firefox：一只美丽的狐狸，最早流行于开发者圈子，逐渐被 Google 带向全世界，将 IE 拉下垄断神坛的第一个对手。近来的 Firefox 已经不输于 Chrome 的速度，并且其传统的强大扩展是很多人都无法离开的原因之一。

Microsoft Edge：微软出品，基于 Chromium。

清理优化

SpaceSniffer：电脑空间管理神器！方便查找占用磁盘空间的文件。

DISM++：Windows 系统优化、清理、维护神器。

DriverStore Explorer：删除无用驱动。

文件复制

FastCopy：复制大量小文件贼快。

下载工具

IDM：多线程下载工具，下百度云很爽。

截图

Snipaste：截图神器。

图片放大

waifu2x-caffe (using CUDA)

waifu2x-converter (using OpenCL)

使用深度卷积神经网络，将噪点和锯齿的部分进行补充，实现图片的低损放大。 中文教程：https://lolico.moe/software/waifu2x.html

手机助手

秋之盒 AutumnBox：开源，简约，免费的安卓手机助手。

Latex 公式

mathpix：获取数学的屏幕截图并将 LaTeX 粘贴到编辑器中，所有这些都只需一个键盘快捷键。

Markdown

Typora：免费，实时预览，所见即所得。（本文即使用 Typora 编辑）

压缩软件

7zip：支持7z, XZ, BZIP2, GZIP, TAR, ZIP and WIM

Bandzip：支持ZIP, 7Z(lzma2), ZIPX(xz), EXE(sfx), TAR, TGZ, LZH(lh7), ISO(joliet), GZ, XZ（推荐）

WinRAR：老牌压缩工具，收费。

火车票辅助

12306 分流抢票：12306 抢火车票辅助工具。
